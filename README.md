# GrooveCube
## _GrooveCube is a casual 3D arcade game where you slide down a track and avoid obsticals. You win by reaching the end of the level. Easy, right?_
### GrooveCube is my first game project. I brought this to life using the Unity game engine. 
### Installed can be found under releases (No admin rights required to install)
